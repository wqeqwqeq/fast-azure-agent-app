# Sub-Agent Config Template

Use this template when generating `app/agent_factory/subagent_config.py`.

## Template

```python
"""Sub-Agent Configuration for Agent Factory.

Generated by /onboard skill.
"""

from .schemas.config import SubAgentConfig, ToolConfig, ToolParameterConfig, UseCaseConfig

# Domain configuration
DOMAIN_NAME = "{domain_name}"
DOMAIN_DESCRIPTION = "{domain_description}"

# Sub-agent definitions
SUB_AGENTS: list[SubAgentConfig] = [
    SubAgentConfig(
        key="{agent_key}",
        name="{agent_name}",
        description="{agent_description}",
        capabilities=[
            "{capability_1}",
            "{capability_2}",
        ],
        tools=[
            ToolConfig(
                name="{tool_name}",
                description="{tool_description}",
                parameters=[
                    ToolParameterConfig(
                        name="{param_name}",
                        type="{param_type}",
                        description="{param_description}",
                        required=True,
                    ),
                ],
            ),
        ],
        use_cases=[
            UseCaseConfig(
                intent="{what user wants}",
                example_queries=["{query1}", "{query2}"],
            ),
        ],
        when_to_use=["{scenario1}", "{scenario2}"],
        when_not_to_use=["{anti_scenario1}"],
    ),
    # ... more agents
]
```

## Field Descriptions

| Field | Description |
|-------|-------------|
| `key` | Unique snake_case identifier (e.g., `leave`, `payroll`) |
| `name` | Display name shown in logs (e.g., `leave-agent`) |
| `description` | One-line description of agent's purpose |
| `capabilities` | List of specific things this agent can do |
| `tools` | List of ToolConfig for each tool function |
| `use_cases` | List of UseCaseConfig with intent and example queries |
| `when_to_use` | Scenarios where triage should route to this agent |
| `when_not_to_use` | Scenarios where another agent is better |

## ToolConfig Fields

| Field | Description |
|-------|-------------|
| `name` | Function name in snake_case (e.g., `get_leave_balance`) |
| `description` | What the tool does |
| `parameters` | List of ToolParameterConfig |

## ToolParameterConfig Fields

| Field | Description |
|-------|-------------|
| `name` | Parameter name |
| `type` | Python type: `str`, `int`, `float`, `bool`, `list`, `dict` |
| `description` | What the parameter is for |
| `required` | Whether parameter is required |

## Example

```python
SUB_AGENTS: list[SubAgentConfig] = [
    SubAgentConfig(
        key="leave",
        name="leave-agent",
        description="Handles vacation and leave requests",
        capabilities=[
            "Check leave balance",
            "Submit leave requests",
            "Cancel leave requests",
        ],
        tools=[
            ToolConfig(
                name="get_leave_balance",
                description="Get remaining PTO/vacation days",
                parameters=[
                    ToolParameterConfig(
                        name="employee_id",
                        type="str",
                        description="Employee ID",
                        required=True,
                    ),
                ],
            ),
            ToolConfig(
                name="submit_leave_request",
                description="Submit a new leave request",
                parameters=[
                    ToolParameterConfig(name="employee_id", type="str", description="Employee ID", required=True),
                    ToolParameterConfig(name="start_date", type="str", description="Start date (YYYY-MM-DD)", required=True),
                    ToolParameterConfig(name="end_date", type="str", description="End date (YYYY-MM-DD)", required=True),
                    ToolParameterConfig(name="leave_type", type="str", description="vacation/sick/personal", required=False),
                ],
            ),
        ],
        use_cases=[
            UseCaseConfig(
                intent="Check balance",
                example_queries=["How many vacation days do I have?", "What's my PTO balance?"],
            ),
            UseCaseConfig(
                intent="Request leave",
                example_queries=["I need time off next week", "Book March 15-22 off"],
            ),
        ],
        when_to_use=["PTO questions", "vacation requests", "sick leave", "time off"],
        when_not_to_use=["Questions about PAY during leave (route to payroll)"],
    ),
]
```
